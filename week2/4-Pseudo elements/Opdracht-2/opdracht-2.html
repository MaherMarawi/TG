Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

  1. ::file-selector-button

  2. ::marker

    3. daarme kan je bijvoorbeeld de punten (markers) van de li tag een opmaak geven

  3. ::placeholder

    4. daarme kan de placeholder van een input of textarea opmaak stijl geven

  4. ::selection

    5. daarme kan de een tekst een opmaak geven wanneer he wordt geselecteerd

  5. ::before

    6. het creëert een pseudo-element dat het eerste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de inhoudseigenschap. Het is standaard inline.

  6. ::first-letter

    7. Selecteert de eerste letter van elk p-element

  7. ::first-line

    8. Selecteert de eerste regel van elk p -element

  8. ::after

    9. het creëert een pseudo-element dat het laatste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de inhoudseigenschap. Het is standaard inline.

  9. ::cue

    10. het creëert een pseudo-element dat het laatste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de inhoudseigenschap. Het is standaard inline.

  10. ::cue-region

    11. het creëert een pseudo-element dat het laatste kind is van het geselecteerde element. Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de inhoudseigenschap. Het is standaard inline.

  11. ::backdrop

    12. Het :: backdrop CSS pseudo-element is een box ter grootte van de viewport die direct wordt weergegeven onder elk element dat in volledig scherm wordt gepresenteerd. Dit omvat zowel elementen die in de modus Volledig scherm zijn geplaatst met behulp van de API voor volledig scherm als

    -elementen.

  12. ::target-text

    13. Het :: target-text CSS pseudo-element vertegenwoordigt de tekst waarnaar is gescrold als de browser scroll-naar-tekstfragmenten ondersteunt. Hiermee kunnen auteurs kiezen hoe ze dat gedeelte van de tekst willen markeren.

  13. ::spelling-error

    14. Het :: spelling-error CSS-pseudo-element vertegenwoordigt een tekstsegment dat door de user-agent is gemarkeerd als onjuist gespeld.

  14. ::grammar-error

    15. The ::grammar-error CSS pseudo-element represents a text segment which the user agent has flagged as grammatically incorrect.
