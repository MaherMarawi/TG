Er zijn verschillende selectors, bijvoorbeeld;
  • type selectors
  • descendant selectors
  • class selectors
    • Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

    - type selectors

    h1 { font-family: sans-serif }

    En dit geldt voor alle h1 in de pagina met 1 css line

    - descendant selectors

    h1 em { color: blue }

    Soms willen auteurs dat selectors overeenkomen met een element dat de afstammeling is van een ander element in de documentboom (bijv. "Overeenkomen met die EM-elementen die deel uitmaken van een H1-element"). Descendant selectors drukken een dergelijke relatie uit in een patroon. Een afstammende selector bestaat uit twee of meer selectors gescheiden door witruimte. Een afstammende selector met de vorm "A B" komt overeen wanneer een element B een willekeurige afstammeling is van een of ander voorouderelement

    - class selectors

    .container {background-color: red}

    een class kun je aan html element of meerdere html elements geven om elkaar samen te stijlen
